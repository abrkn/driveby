# driveby

Replay Bitcoin Core (BTC) transactions onto DriveNet

## Install

`npm install`

## Configuration

Define `.env` file:

```
SOURCE_BITCOIND_URL=bitcoin://user:xxx@1.2.3.4:8332
DESTINATION_BITCOIND_URL=bitcoin://user:xxx@1.2.3.4:8332
```

## Running

`npm start`
